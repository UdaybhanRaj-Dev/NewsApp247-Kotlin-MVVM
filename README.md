# 📰 NewsApp247 - Kotlin MVVM News App

NewsApp247 is a modern Android News Application built using **Kotlin** and the **MVVM (Model-View-ViewModel)** architecture.  
The app fetches real-time news articles from a public News API and displays them in a clean and user-friendly interface.

---

## 🚀 Features

- 📰 Real-time News using API Integration  
- 📂 Category-wise News (Technology, Sports, Business, etc.)  
- 🔎 Search News Articles  
- 💾 Offline Caching Support  
- 📱 Clean & Responsive UI  
- 🧠 MVVM Architecture Implementation  
- ⚡ Coroutines for Background Processing  

---

## 🏗️ Architecture

This project follows the **MVVM Architecture Pattern**:

- Model → Data Layer (API + Repository)
- View → UI (Activity/Fragment)
- ViewModel → Business Logic + LiveData

This improves:
- Code Maintainability  
- Scalability  
- Separation of Concerns  

---

## 🛠️ Tech Stack

- **Language:** Kotlin  
- **Architecture:** MVVM  
- **Networking:** Retrofit  
- **JSON Parsing:** Gson  
- **Image Loading:** Glide  
- **Async:** Kotlin Coroutines  
- **Live Data:** LiveData & ViewModel  

---

## 🔌 API Used

This app uses a public News API to fetch latest headlines.

> ⚠️ Note: Add your API key inside the Constants file before running the app.
